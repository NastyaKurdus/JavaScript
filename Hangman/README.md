Для запуску:
Достатньо запустити файл Hangman.html.
Опис функціоналу:
Після запуску з'явиться игрове поле на якому розміщені кнопки-літери,комірки з "_" на початку гри ,а  після натискання на кнопку з правильною літерою в ній з'явиться ця літера.Також присутні 2 кнопки:"Show word" та "Help".Перша при натисканні покаже все слово,але це буде означати автоматичний програш.Друга при натисканні відкриє будь-яку литеру в слові,але за це відбере один хід (цю кнопку можна використати лише 2 рази).Якщо гравець вгадав слово, то з'явиться повідомлення про перемогу і кнопка "Next word",якщо програв, то з'явиться повідомлення про поразку і кнопка "Try again".
